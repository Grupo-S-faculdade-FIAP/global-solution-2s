## 📜 Descrição

Esta pasta reúne os arquivos e experimentos desenvolvidos além do escopo mínimo da entrega principal do TerraSense — Global Solution.

Aqui estão documentados aprofundamentos, experimentos exploratórios, protótipos e ideias que enriquecem a solução principal mas que vão além dos requisitos obrigatórios:

- Experimentos com modelos alternativos para detecção de queimadas
- Extensões do dashboard com novas visualizações
- Integrações extras com APIs ambientais
- Protótipos de alertas em tempo real
- Exploração de datasets complementares

---

## 📁 Estrutura

Os arquivos "Ir Além" estão organizados por módulo/tema, seguindo a mesma estrutura modular do projeto principal:

```
Ir Além/
├── README.md         ← este arquivo
├── notebooks/        ← experimentos exploratórios (Jupyter)
├── scripts/          ← scripts de prototipagem
└── docs/             ← documentação de experimentos extras
```
